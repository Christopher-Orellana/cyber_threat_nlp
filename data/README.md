# Data

The raw dataset is not included in this repository.

Expected local file path:

`data/TheHackerNews_Dataset.xlsx`

This project expects an Excel file containing cybersecurity article text and threat-category labels for NLP classification.

Dataset source:

`https://github.com/cypher-07/Cybersecurity-News-Article-Dataset`

Expected columns may include:

- article text or cleaned text
- threat category / label

The dataset is excluded from version control to avoid uploading raw or potentially copyrighted source data.
